Fixed Asset PDF Report

This App produces Fixed Asset Schedule and Fixed Asset Register Report.



This app will create few additional fields in Asset under Accounting and it will also produce standard Fixed Asset Reports: Fixed Asset Schedule and Fixed Asset Register Report in PDF format.


Dependencies
  • Accounting
  • Asset Management

Workflow Overview

Create Fixed Asset Entries.




To generate Report, simply go to Accounting>Reports> Fixed Asset Report and follow the Wizard to select Date and Month. Select Date Range for Fixed Asset Schedule and Select Month for Fixed Asset Register.
